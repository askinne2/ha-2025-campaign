## 🎯 Interactive States

### Story Card States
```
Normal:    Border: Light Gray, No shadow
Hover:     Border: Catalyst Red, Lifted 5px, Shadow
Focus:     Catalyst Red outline (keyboard nav)
Active:    Slightly pressed appearance
```

### Story Badge States
```
Normal:    Colored background, white text
Hover:     Scale(1.05), Enhanced shadow
Focus:     2px outline (keyboard nav)
Active:    Slight scale down
```

## 💡 Usage Tips

### For Content Managers
1. **Update links** when story pages go live
2. **Add images** to story cards (optional enhancement)
3. **Track engagement** via analytics on story links
4. **A/B test** different teaser quotes

### For Developers
1. **Story pages** use same template structure
2. **CSS variables** make color updates easy
3. **Responsive breakpoints** match site-wide standards
4. **Print styles** ensure clean PDFs

### For Campaign Team
1. **Promote stories** on social media matching calendar dates
2. **Email teasers** before story publication
3. **Share quotes** from stories in donor communications
4. **Track metrics** on story engagement vs donations
